# Azure Static Website Hosting Project

This project demonstrates how to deploy a publicly accessible static website using **Azure Storage Static Website hosting**. The project showcases foundational cloud engineering concepts including object storage, resource organization, and cloud-based web hosting.

The website is hosted directly from **Azure Blob Storage** without requiring traditional web servers.
